<h1 align="center">🎓 My Hugging Face NLP Study Dashboard</h1>
<h3 align="center">Interactive NLP Toolkit using Transformers & Gradio</h3>

<hr/>

<h2>🚀 Project Overview</h2>
<p>
This project is an interactive NLP dashboard built using 
Hugging Face Transformers and Gradio.
</p>

<p>
It provides multiple Natural Language Processing tools 
in a single unified web interface.
</p>

<hr/>

<h2>🧠 Features</h2>

<ul>
<li>✍️ Text Generation</li>
<li>📝 Text Summarization</li>
<li>😊 Sentiment Analysis</li>
<li>🌍 English → Arabic Translation</li>
<li>❓ Question Answering</li>
</ul>

<hr/>

<h2>🖥️ Interface Screens</h2>
